Rishikesh Context

Location, nearby landmarks, and what this ideation is meant to give.

Rishikesh, Uttarakhand

Set in the Himalayan foothills along the Ganges, Rishikesh blends spiritual calm, river energy, traveler culture, yoga, craft, and café life in a way that naturally supports multiple design moods.

This ideation translates that setting into four usable interior directions, so the cafe can feel rooted in place while still offering distinct guest experiences.

What The Ideation Gives

A moodboard-led concept system for the cafe: one retro, one boho, one Indian craft-forward, and one moto-industrial. Each board turns visual references into a clearer design language, palette, and atmosphere.
